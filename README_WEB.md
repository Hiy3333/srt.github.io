# 🌐 SRT 자막 웹 번역기

Flask 기반 웹 인터페이스로 SRT 자막 파일을 여러 언어로 번역하는 서비스입니다.

## ✨ 웹 버전 특징

- 🌐 **웹 브라우저에서 바로 사용** - 프로그램 설치 불필요
- 📁 **드래그 앤 드롭 지원** - 파일을 끌어다 놓기만 하면 됩니다
- ✅ **다중 언어 선택** - 체크박스로 원하는 언어만 선택
- 📦 **ZIP 다운로드** - 번역된 모든 파일을 한 번에 다운로드
- 🎨 **모던한 UI** - 깔끔하고 직관적인 디자인
- 📱 **반응형 디자인** - 모바일, 태블릿, 데스크톱 모두 지원

## 🚀 빠른 시작

### 1단계: 패키지 설치

```bash
pip install -r requirements_web.txt
```

### 2단계: 웹 서버 실행

```bash
python app.py
```

또는 Windows에서:
```bash
run_web.bat
```

### 3단계: 브라우저 접속

웹 브라우저에서 다음 주소로 접속하세요:
```
http://localhost:5000
```

## 📖 사용 방법

### 웹 인터페이스 사용

1. **API 키 입력**
   - OpenAI API 키를 입력 필드에 붙여넣기
   - 키는 [OpenAI Platform](https://platform.openai.com/api-keys)에서 발급

2. **SRT 파일 업로드**
   - "파일 선택" 클릭하여 파일 선택
   - 또는 파일을 드래그 앤 드롭

3. **언어 선택**
   - 번역하고 싶은 언어 체크박스 선택
   - 여러 개 선택 가능
   - "전체 선택" / "전체 해제" 버튼 활용

4. **번역 시작**
   - "🚀 번역 시작" 버튼 클릭
   - 진행 상황이 화면에 표시됩니다

5. **다운로드**
   - 번역 완료 시 자동으로 ZIP 파일 다운로드
   - ZIP 파일에는 선택한 모든 언어의 SRT 파일 포함

## 🌍 지원 언어

| 언어 | 코드 | 출력 파일명 |
|------|------|-------------|
| 영어 | en | filename_en.srt |
| 일본어 | ja | filename_ja.srt |
| 태국어 | th | filename_th.srt |
| 중국어 | zh | filename_zh.srt |
| 인도네시아어 | id | filename_id.srt |
| 스페인어 | es | filename_es.srt |

## 🔧 고급 설정

### 포트 변경

`app.py` 파일의 마지막 줄을 수정하세요:

```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 포트를 원하는 번호로 변경
```

### 최대 파일 크기 변경

`app.py` 파일의 설정을 수정하세요:

```python
app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024  # 16MB (원하는 크기로 변경)
```

### 외부 접속 허용

로컬 네트워크에서 다른 기기로 접속하려면:

1. 현재 PC의 IP 주소 확인 (예: 192.168.0.10)
2. 다른 기기에서 `http://192.168.0.10:5000` 접속

**주의**: 방화벽 설정이 필요할 수 있습니다.

## 🔒 보안 및 개인정보

- **API 키 보안**: 입력한 API 키는 서버에 저장되지 않습니다
- **파일 처리**: 업로드된 파일은 메모리에서만 처리되며 서버에 저장되지 않습니다
- **HTTPS 권장**: 프로덕션 환경에서는 HTTPS를 사용하세요

## 💰 비용 안내

OpenAI API 사용 비용 (GPT-4o-mini 기준):
- 입력 토큰: $0.150 / 1M 토큰

## ⚠️ 주의사항

1. **API 키 관리**
   - API 키를 타인과 공유하지 마세요
   - 공개된 곳에 키를 노출하지 마세요

   - API 호출 시간은 네트워크 속도에 따라 다름

4. **브라우저 호환성**
   - 최신 버전의 Chrome, Firefox, Safari, Edge 권장

## 🐛 문제 해결

### 서버가 시작되지 않음
```bash
# 포트가 이미 사용 중인 경우
netstat -ano | findstr :5000  # Windows
lsof -i :5000                 # Mac/Linux
```

### "ModuleNotFoundError" 오류
```bash
pip install -r requirements_web.txt
```

### API 키 오류
- API 키가 올바른지 확인
- [사용량 페이지](https://platform.openai.com/usage)에서 크레딧 확인
- API 키 권한 확인

### 번역이 너무 느림
- 인터넷 연결 상태 확인
- 자막 블록 수가 많으면 시간이 오래 걸립니다
- 한 번에 적은 수의 언어만 선택해보세요

### 다운로드 실패
- 팝업 차단 해제
- 브라우저 다운로드 권한 확인
- 충분한 디스크 공간 확인
