# 📄 SRT 자막 번역기 - 순수 HTML 버전

서버 설치 없이 브라우저에서 바로 실행되는 SRT 자막 번역기입니다!

## 🌟 특징

- ✅ **서버 불필요** - 파일 하나만 브라우저에서 열면 끝!
- ✅ **설치 불필요** - Python, Flask 등 아무것도 설치 안 해도 됨
- ✅ **완전한 개인정보 보호** - 모든 처리가 브라우저에서만 이루어짐
- ✅ **드래그 앤 드롭** - 파일을 끌어다 놓기만 하면 됩니다
- ✅ **다중 언어 선택** - 원하는 언어만 체크박스로 선택
- ✅ **ZIP 다운로드** - 번역된 모든 파일을 한 번에 다운로드
- ✅ **실시간 진행률** - 번역 진행 상황을 실시간으로 확인
- ✅ **모던한 디자인** - 깔끔하고 직관적인 UI
- ✅ **반응형** - PC, 태블릿, 모바일 모두 지원

## 🚀 사용 방법

### 방법 1: 로컬에서 실행 (가장 간단!)

1. **파일 열기**
   - `srt_translator.html` 파일을 더블클릭
   - 또는 브라우저로 파일을 드래그 앤 드롭

2. **번역 시작**
   - OpenAI API 키 입력
   - SRT 파일 업로드 (드래그 앤 드롭 가능)
   - 원하는 언어 선택
   - "🚀 번역 시작" 버튼 클릭

3. **다운로드**
   - 번역 완료 시 자동으로 ZIP 파일 다운로드
   - ZIP 파일 압축 해제하면 번역된 SRT 파일들이 있습니다

### 방법 2: 온라인에서 공유

파일을 웹 호스팅에 올려서 다른 사람들과 공유할 수 있습니다:
- GitHub Pages
- Netlify
- Vercel
- 기타 정적 호스팅 서비스

## 📖 상세 사용 가이드

### 1단계: API 키 준비

1. [OpenAI 플랫폼](https://platform.openai.com/api-keys) 접속
2. 로그인 또는 회원가입
3. "Create new secret key" 클릭
4. API 키 복사 (sk-로 시작하는 키)

### 2단계: 파일 열기

**Windows:**
```
srt_translator.html 더블클릭
```

**Mac:**
```
srt_translator.html을 Safari나 Chrome으로 열기
```

**Linux:**
```bash
xdg-open srt_translator.html
# 또는
firefox srt_translator.html
```

### 3단계: 번역 실행

1. **API 키 입력**
   - 복사한 OpenAI API 키를 입력 필드에 붙여넣기
   - 키는 브라우저에만 저장되며 서버로 전송되지 않습니다

2. **SRT 파일 업로드**
   - "파일 선택" 버튼 클릭하여 파일 선택
   - 또는 파일을 화면으로 드래그 앤 드롭

3. **언어 선택**
   - 번역하고 싶은 언어 체크박스 선택
   - 전체 선택 버튼으로 한 번에 선택 가능
   - 지원 언어:
     - ✅ English (영어)
     - ✅ Japanese (일본어)
     - ✅ Thai (태국어)
     - ✅ Chinese (중국어)
     - ✅ Indonesian (인도네시아어)
     - ✅ Spanish (스페인어)

4. **번역 시작**
   - "🚀 번역 시작" 버튼 클릭
   - 진행 상황이 실시간으로 표시됩니다
   - 언어별, 블록별 진행률 확인 가능

5. **결과 다운로드**
   - 번역 완료 시 자동으로 ZIP 파일 다운로드
   - ZIP 파일명: `원본파일명_translated_날짜시간.zip`
   - ZIP 압축 해제 시 포함된 파일:
     - `원본파일명_en.srt` (영어)
     - `원본파일명_ja.srt` (일본어)
     - `원본파일명_th.srt` (태국어)
     - `원본파일명_zh.srt` (중국어)
     - `원본파일명_id.srt` (인도네시아어)
     - `원본파일명_es.srt` (스페인어)

## 🔒 보안 및 개인정보

### 완전한 클라이언트 사이드 처리

이 HTML 파일은 **브라우저에서만** 모든 작업을 처리합니다:

1. **파일 처리**: 업로드한 SRT 파일은 브라우저 메모리에만 로드

3. **번역 요청**: 브라우저에서 OpenAI API로 직접 요청

- API 키는 localStorage에 저장되지 않습니다
- 페이지를 새로고침하면 API 키는 사라집니다
- API 키는 OpenAI API 호출 시에만 사용됩니다

## 🐛 문제 해결

### CORS 오류
일부 브라우저에서는 로컬 파일 열기 시 제한이 있을 수 있습니다.

**해결 방법:**
- Chrome에서 열기 (권장)
- 또는 간단한 로컬 서버 실행:
  ```bash
  # Python 3
  python -m http.server 8000
  
  # 그 다음 브라우저에서
  http://localhost:8000/srt_translator.html
  ```

### API 키 오류
```
Error: Incorrect API key provided
```

**해결 방법:**
- API 키가 올바른지 확인
- sk-로 시작하는지 확인
- [사용량 페이지](https://platform.openai.com/usage)에서 크레딧 확인

### 번역이 느림
- 자막 블록 수가 많으면 시간이 오래 걸립니다
- 한 번에 1-2개 언어만 선택해보세요
- 인터넷 연결 상태를 확인하세요

### 다운로드 안 됨
- 팝업 차단 해제
- 브라우저 다운로드 권한 확인
- 다운로드 폴더 용량 확인

### SRT 파일 파싱 오류
```
SRT 파일을 파싱할 수 없습니다
```

**해결 방법:**
- 파일이 UTF-8 인코딩인지 확인
- SRT 형식이 올바른지 확인:
  ```
  1
  00:00:01,000 --> 00:00:03,000
  첫 번째 자막

  2
  00:00:03,500 --> 00:00:06,000
  두 번째 자막
  ```
