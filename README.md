# SRT 자막 파일 다국어 번역 프로그램

GPT API를 사용하여 한글 SRT 자막 파일을 여러 언어로 자동 번역하는 프로그램입니다.

## 🌏 지원 언어

- 🇬🇧 영어 (English)
- 🇯🇵 일본어 (Japanese)
- 🇹🇭 태국어 (Thai)
- 🇨🇳 중국어 간체 (Chinese Simplified)
- 🇮🇩 인도네시아어 (Indonesian)
- 🇪🇸 스페인어 (Spanish)

## 📋 주요 기능

- ✅ SRT 파일 형식 완벽 지원 (블록번호, 타임코드, 대사)
- ✅ GPT-4o-mini 모델을 사용한 고품질 번역
- ✅ 각 언어별 별도 SRT 파일 자동 생성
- ✅ 실시간 진행률 표시
- ✅ 여러 줄 대사 지원
- ✅ UTF-8 인코딩으로 모든 언어 지원

## 🚀 설치 방법

### 1. Python 설치 확인
Python 3.7 이상이 필요합니다.
```bash
python --version
```

### 2. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

## 🔑 OpenAI API 키 발급

1. [OpenAI 웹사이트](https://platform.openai.com/)에 접속
2. 계정 생성 또는 로그인
3. API Keys 섹션으로 이동
4. "Create new secret key" 클릭하여 API 키 생성
5. 생성된 키를 안전한 곳에 보관

## 📁 출력 파일

입력 파일: `example.srt`

생성되는 출력 파일들:
- `example_en.srt` - 영어 번역
- `example_ja.srt` - 일본어 번역
- `example_th.srt` - 태국어 번역
- `example_zh.srt` - 중국어 번역
- `example_id.srt` - 인도네시아어 번역
- `example_es.srt` - 스페인어 번역

## 📝 SRT 파일 형식

SRT 파일은 다음과 같은 형식을 따릅니다:

```
1
00:00:01,000 --> 00:00:03,000
안녕하세요, 여러분!

2
00:00:03,500 --> 00:00:06,000
오늘은 SRT 번역 프로그램을 소개합니다.
```

각 블록은:
1. 블록 번호
2. 타임코드 (시작 --> 끝)
3. 대사 텍스트
4. 빈 줄로 구분

## ⚙️ 고급 설정

### 다른 GPT 모델 사용

`srt_translator.py` 파일의 94번째 줄을 수정하여 다른 모델을 사용할 수 있습니다:

### Temperature 조정

번역의 창의성을 조정하려면 99번째 줄을 수정하세요:

```python
temperature=0.3,  # 0.0-1.0 사이 값 (낮을수록 일관성 있음)
```

## 🐛 문제 해결

### "ModuleNotFoundError: No module named 'openai'"
```bash
pip install openai
```

### "API 키 오류"
- API 키가 올바른지 확인
- [OpenAI 사용량 페이지](https://platform.openai.com/usage)에서 크레딧 잔액 확인

### "파일을 찾을 수 없습니다"
- 파일 경로가 올바른지 확인
- 절대 경로 또는 상대 경로 사용
- Windows에서는 백슬래시(\\) 또는 슬래시(/) 모두 사용 가능

### "인코딩 오류"
- SRT 파일을 UTF-8 인코딩으로 저장
- 메모장에서: 다른 이름으로 저장 → 인코딩: UTF-8
